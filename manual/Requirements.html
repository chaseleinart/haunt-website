2.2 Requirements

Haunt depends on the following packages:

  • GNU Guile version 2.0.11 or later

The following dependencies are optional:

  • Guile-Reader version 0.6 or later (for Skribe support)
  • guile-commonmark version 0.1 or later (for CommonMark support)
